GEMINI API SETUP (FOR HF SPACES FALLBACK)
═══════════════════════════════════════════════════════════════════════════════

📝 If Ollama is slow on HF Spaces, enable Gemini:

   3. Click "Create API Key"
   4. Copy the key

Step 2: Add to HF Space
   1. Go to: https://huggingface.co/spaces/nachikethshetty/ai-research-assistant-openenv
   2. Click Settings (gear icon, top right)
   3. Go to "Repository secrets"
   4. Add new secret:

      Value: (paste your key)
   5. Save

✅ Now your app will auto-fallback to Gemini if Ollama unavailable!
