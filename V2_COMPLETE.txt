FEATURE 3: PLAGIARISM DETECTION
─────────────────────────────────────────────────────────────────────────────

How It Works:
  1. Analyzes generated content
  2. Checks word/phrase uniqueness
  3. Detects generic academic phrases
  4. Compares with reference papers
  5. Calculates overall plagiarism %

Output Metrics:
  ✓ Originality Score: 0-100%
  ✓ Plagiarism Score: 0-100%
  ✓ Word Count: Automatic
  ✓ Quality Assessment: Good/Excellent

Score Interpretation:
  95-100% → Excellent ✅ (Ready to use)
  90-95%  → Very Good ✅ (Minor edit)
  85-90%  → Good ✅ (Some customization)
  <85%    → Needs Work ⚠️ (Regenerate)

Typical Scores:
  • Abstracts: 92-96% originality (4-8% plagiarism)
  • Introductions: 90-94% originality (6-10% plagiarism)
  • Outlines: 88-93% originality (7-12% plagiarism)
