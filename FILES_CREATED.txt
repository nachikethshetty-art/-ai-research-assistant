RAG LAYER:
  └─ pipeline.py: Query → papers → answer + gaps

RL LAYER:
  └─ feedback_system.py: Tracks feedback → updates weights

  └─ research_generator.py: Topic → abstract/intro with plagiarism score

  ├─ streamlit_app.py: Interactive UI dashboard
  └─ api/server.py: REST endpoints for integration

AUTOMATION LAYER:
